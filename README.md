# 📚 LiterAlura

Catálogo de libros desarrollado en **Java con Spring Boot** que consume la API pública de **Gutendex** para buscar y mostrar información de libros y autores desde la consola.

Este proyecto permite realizar consultas sobre libros utilizando una interfaz de texto interactiva.

---

## 🚀 Funcionalidades

El programa permite al usuario interactuar mediante un menú en consola con las siguientes opciones:

1. Buscar libro por título  
2. Listar libros más descargados  
3. Buscar libros por idioma  
4. Buscar libros por autor  

Cada búsqueda muestra información como:

- 📖 **Título del libro**
- ✍️ **Autor**
- 🌎 **Idioma**
- ⬇️ **Número de descargas**

---

## 🧰 Tecnologías utilizadas

- **Java 21**
- **Spring Boot**
- **Maven**
- **HttpClient**
- **Jackson (JSON Parser)**
- **API Gutendex**
